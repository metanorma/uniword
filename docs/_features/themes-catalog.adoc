= Themes Catalog

Uniword bundles 29 open-source themes that replace the proprietary Microsoft Office themes. Each theme is defined in `data/themes/*.yml` and can be applied to any document.

== Using Themes

[source,ruby]
----
doc = Uniword::Document.new
doc.apply_theme('celestial')
----

The `apply_theme` method looks up the theme name in the bundled catalog and applies the complete theme definition (colors, fonts, and formatting) to the document.

== Theme Detection

When loading a DOCX file created with Microsoft Word, Uniword can detect the original MS theme from its color fingerprint and automatically map it to the corresponding Uniword theme. The mapping is defined in `config/theme_mapping.yml`.

== Theme Structure

Each theme YAML file contains:

* **Color scheme** -- 12 theme colors (dk1, dk2, lt1, lt2, accent1-accent6)
* **Font scheme** -- Heading and body font families
* **Format scheme** -- Fill, line, and effect definitions

All theme definitions use OFL-licensed fonts and open color palettes.
