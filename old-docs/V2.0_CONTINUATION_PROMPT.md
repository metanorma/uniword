## Critical Principles

### 1. ZERO RAW XML Tolerance

**NO** `raw_*` attributes, **NO** `UnknownElement`, **NO** escape hatches.

Every OOXML element MUST be a proper lutaml-model class.

### 2. Pattern 0 is Sacred

Attributes MUST be declared BEFORE xml mappings. The generator enforces this.

### 3. Schema-Driven Everything

No hardcoding. Everything in YAML schemas, code generated automatically.

### 4. MECE Architecture

- Each element class has ONE responsibility
- Separation of concerns strictly maintained
- Inheritance and composition over duplication

### 5. Object-Oriented Design

- Model-driven, not serializer-driven
- Document structure is pure Ruby objects
- Serialization is separate layer

1. **Does this element exist in ISO 29500 spec?**
   - If yes: Model it properly
   - If no: Don't invent it

2. **Can this be generated from schema?**
   - If yes: Add to schema
