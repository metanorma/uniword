- **Features**:
  - Holds Plurimath formula objects
  - Supports inline and block display modes
  - Configurable alignment (left, center, right)
  - Line breaking support
  - Multi-format conversion (LaTeX, MathML, AsciiMath, OMML)
- **API Methods**:
  - `from_omml(xml)` - Parse OMML to equation
  - `to_omml(options)` - Serialize equation to OMML
  - `to_latex()` - Convert to LaTeX
  - `to_mathml()` - Convert to MathML
  - `to_asciimath()` - Convert to AsciiMath

- **Lines**: 158

#### PlurimathAdapter (`lib/uniword/math/plurimath_adapter.rb`)
- **Purpose**: Bidirectional conversion between OMML and Plurimath
- **Responsibilities**:
  - Parse OMML XML to Plurimath formulas
  - Serialize Plurimath formulas to OMML XML
  - Extract equation properties (alignment, display type, breaks)
  - Handle display modes (inline vs block)
- **Key Methods**:
  - `from_omml(omml_xml)` - Convert OMML to MathEquation
  - `to_omml(equation, options)` - Convert MathEquation to OMML
  - `parse_omml_to_formula(node)` - Parse using Plurimath
  - `formula_to_omml(formula)` - Serialize using Plurimath
  - `extract_properties(node)` - Extract OMML properties
- **Features**:
  - Graceful error handling
  - Namespace-aware XML processing
  - Pretty-print support
  - Property preservation

## API Examples

### Creating Math Equations

```ruby
# From LaTeX
para = Uniword::Paragraph.new
para.add_math_equation("x^2 + y^2 = z^2", format: :latex)

# From MathML
para.add_math_equation("<math><mi>x</mi></math>", format: :mathml)

# Block equation with alignment
para.add_math_equation(
  "\\int_{0}^{\\infty} e^{-x^2} dx",
  format: :latex,
  display_type: :block,
  alignment: :center
)

# With Plurimath formula object
formula = Plurimath::Math.parse("x^2", :latex)
equation = Uniword::MathEquation.new(formula: formula)
para.add_run(equation)
```

### Converting Formats

```ruby
equation = para.math_equations.first

# Convert to different formats
latex = equation.to_latex       # "x^2 + y^2 = z^2"
mathml = equation.to_mathml     # "<math>...</math>"
ascii = equation.to_asciimath   # "x^2 + y^2 = z^2"
omml = equation.to_omml         # "<m:oMath>...</m:oMath>"
```

### Reading from DOCX

```ruby
# Parse OMML from document
omml_xml = '<m:oMath>...</m:oMath>'
equation = Uniword::MathEquation.from_omml(omml_xml)

# Access formula
puts equation.to_latex
puts equation.inline? # true/false
```
