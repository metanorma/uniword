# Uniword v3.0 Plurimath Integration Architecture

## Executive Summary

**Goal:** Integrate Plurimath library for comprehensive mathematical equation support in DOCX documents.

**Approach:** Adapter pattern with Plurimath as external dependency

**Target Version:** v3.0.0 (after v2.0 schema expansion complete)

**Plurimath Capabilities:**
- Parse and convert between: MathML, AsciiMath, LaTeX, UnicodeMath, OMML
- Round-trip conversions with formula preservation
- Number formatting support
- Complete Office Math Markup Language (OMML) support

## Current State Analysis

## v3.0 Architecture Design

### Design Principles

1. **Adapter Pattern:** Plurimath is external dependency, not copied code
2. **Separation of Concerns:** Math handling is separate from other OOXML elements
3. **Lazy Loading:** Plurimath loaded only when math operations needed
4. **External Configuration:** Math settings in YAML configuration files
5. **Format Preservation:** Round-trip OOXML ↔ Plurimath without loss

## Class Design

### MathEquation Element

```ruby
# lib/uniword/math_equation.rb

module Uniword
  # Represents a mathematical equation in a document.
  #
  # Responsibility: Model a math equation element that can be serialized
  # to/from OOXML and converted to various math formats via Plurimath.
  #
  # This class uses the Adapter pattern to integrate with Plurimath
  # without tight coupling.
  #
  # @example Creating a math equation from OMML
  #   omml = '<m:oMath>...</m:oMath>'
  #   equation = MathEquation.new(omml: omml)
  #
  # @example Converting to LaTeX
  #   latex = equation.to_latex
  #
  # @example Round-trip conversion
  #   omml_original = '<m:oMath>...</m:oMath>'
  #   equation = MathEquation.new(omml: omml_original)
  #   omml_regenerated = equation.to_omml
  #
  class MathEquation < Element
    attr_accessor :formula  # Plurimath::Math::Formula object
    attr_accessor :display_mode  # :inline or :block
    attr_accessor :alignment  # :left, :center, :right

    # Initialize math equation
    #
    # @param omml [String, nil] OMML XML string
    # @param formula [Plurimath::Math::Formula, nil] Plurimath formula object
    # @param display_mode [Symbol] :inline or :block (default: :inline)
    # @param alignment [Symbol] :left, :center, :right (default: :left)
    #
    # @example From OMML
    #   equation = MathEquation.new(omml: '<m:oMath>...</m:oMath>')
    #
    # @example From Plurimath formula
    #   formula = Plurimath::Math.parse('x^2 + y^2', :latex)
    #   equation = MathEquation.new(formula: formula)
    #
    def initialize(omml: nil, formula: nil, display_mode: :inline, alignment: :left)
      super()

      @display_mode = display_mode
      @alignment = alignment

      if omml
        # Parse OMML to Plurimath formula using adapter
        @formula = PlurimathAdapter.from_omml(omml)
      elsif formula
        # Use provided Plurimath formula directly
        @formula = formula
      end
    end

    # Convert to OMML (OOXML format)
    #
    # @param options [Hash] Conversion options
    # @option options [Boolean] :display_style Use display style
    # @return [String] OMML XML string
    #
    # @example Convert to OMML
    #   omml_xml = equation.to_omml
    #
    def to_omml(options = {})
      return nil unless @formula

      # Use adapter to convert Plurimath formula to OMML
      PlurimathAdapter.to_omml(@formula, options.merge(
        display_mode: @display_mode,
        alignment: @alignment
      ))
    end

    # Convert to LaTeX
    #
    # @param options [Hash] Conversion options
    # @return [String] LaTeX math string
    #
    # @example Convert to LaTeX
    #   latex = equation.to_latex
    #   # => '\frac{x^2 + y^2}{2}'
    #
    def to_latex(options = {})
      return nil unless @formula
      @formula.to_latex(options)
    end

    # Convert to MathML
    #
    # @param options [Hash] Conversion options
    # @return [String] MathML XML string
    #
    def to_mathml(options = {})
      return nil unless @formula
      @formula.to_mathml(options)
    end

    # Convert to AsciiMath
    #
    # @param options [Hash] Conversion options
    # @return [String] AsciiMath string
    #
    def to_asciimath(options = {})
      return nil unless @formula
      @formula.to_asciimath(options)
    end

    # Convert to UnicodeMath
    #
    # @param options [Hash] Conversion options
    # @return [String] UnicodeMath string
    #
    def to_unicodemath(options = {})
      return nil unless @formula
      @formula.to_unicodemath(options)
    end

    # Get displayable representation
    #
    # @param format [Symbol] Output format (:latex, :mathml, :asciimath, :unicodemath)
    # @return [String] Formatted math string
    #
    def display_as(format = :latex)
      case format
      when :latex then to_latex
      when :mathml then to_mathml
      when :asciimath then to_asciimath
      when :unicodemath then to_unicodemath
      when :omml then to_omml
      else
        raise ArgumentError, "Unknown format: #{format}"
      end
    end

    # Check if equation is block-level (display math)
    #
    # @return [Boolean] true if block-level
    #
    def block_level?
      @display_mode == :block
    end

    # Check if equation is inline
    #
    # @return [Boolean] true if inline
    #
    def inline?
      @display_mode == :inline
    end
  end
end
```

### PlurimathAdapter

```ruby
# lib/uniword/math/plurimath_adapter.rb

module Uniword
  module Math
    # Adapter for Plurimath library integration.
    #
    # Responsibility: Convert between Uniword's MathEquation representation
    # and Plurimath's Formula representation. Encapsulates all Plurimath
    # library interactions.
    #
    # This adapter implements the Adapter pattern to decouple Uniword from
    # Plurimath library specifics, making it easier to swap or upgrade the
    # math library in the future if needed.
    #
    # @example Parse OMML to Plurimath
    #   formula = PlurimathAdapter.from_omml('<m:oMath>...</m:oMath>')
    #
    # @example Convert Plurimath to OMML
    #   omml = PlurimathAdapter.to_omml(formula)
    #
    class PlurimathAdapter
      # Load Plurimath library (lazy loading)
      #
      # @return [void]
      # @raise [LoadError] if Plurimath gem not installed
      #
      def self.ensure_plurimath_loaded
        return if defined?(Plurimath)

        begin
          require 'plurimath'
        rescue LoadError => e
          raise LoadError,
                "Plurimath gem is required for math equation support. " \
                "Add 'gem \"plurimath\"' to your Gemfile and run 'bundle install'."
        end
      end

      # Parse OMML to Plurimath Formula
      #
      # @param omml_string [String] OMML XML string
      # @return [Plurimath::Math::Formula] Parsed formula
      # @raise [ArgumentError] if omml_string is invalid
      #
      # @example Parse OMML
      #   omml = '<m:oMath><m:f><m:num><m:r><m:t>x</m:t></m:r></m:num>...</m:f></m:oMath>'
      #   formula = PlurimathAdapter.from_omml(omml)
      #
      def self.from_omml(omml_string)
        ensure_plurimath_loaded

        raise ArgumentError, "OMML string cannot be nil" if omml_string.nil?
        raise ArgumentError, "OMML string cannot be empty" if omml_string.empty?

        # Parse OMML using Plurimath
        Plurimath::Math.parse(omml_string, :omml)
      rescue => e
        raise ArgumentError, "Failed to parse OMML: #{e.message}"
      end

      # Convert Plurimath Formula to OMML
      #
      # @param formula [Plurimath::Math::Formula] Plurimath formula object
      # @param options [Hash] Conversion options
      # @option options [Boolean] :display_style Use display style (default: true)
      # @option options [Symbol] :display_mode :inline or :block
      # @option options [Symbol] :alignment :left, :center, :right
      # @return [String] OMML XML string
      #
      # @example Convert to OMML
      #   formula = Plurimath::Math.parse('x^2 + y^2', :latex)
      #   omml = PlurimathAdapter.to_omml(formula)
      #
      def self.to_omml(formula, options = {})
        ensure_plurimath_loaded

        raise ArgumentError, "Formula cannot be nil" if formula.nil?
        raise ArgumentError, "Formula must be a Plurimath::Math::Formula" unless
          formula.is_a?(Plurimath::Math::Formula)

        # Convert to OMML using Plurimath
        display_style = options.fetch(:display_style, true)
        formula.to_omml(display_style: display_style)
      rescue => e
        raise ArgumentError, "Failed to convert to OMML: #{e.message}"
      end

      # Parse LaTeX to Plurimath Formula
      #
      # @param latex_string [String] LaTeX math string
      # @return [Plurimath::Math::Formula] Parsed formula
      #
      def self.from_latex(latex_string)
        ensure_plurimath_loaded
        Plurimath::Math.parse(latex_string, :latex)
      end

      # Parse MathML to Plurimath Formula
      #
      # @param mathml_string [String] MathML XML string
      # @return [Plurimath::Math::Formula] Parsed formula
      #
      def self.from_mathml(mathml_string)
        ensure_plurimath_loaded
        Plurimath::Math.parse(mathml_string, :mathml)
      end

      # Parse AsciiMath to Plurimath Formula
      #
      # @param asciimath_string [String] AsciiMath string
      # @return [Plurimath::Math::Formula] Parsed formula
      #
      def self.from_asciimath(asciimath_string)
        ensure_plurimath_loaded
        Plurimath::Math.parse(asciimath_string, :asciimath)
      end

      # Check if Plurimath is available
      #
      # @return [Boolean] true if Plurimath gem is loaded
      #
      def self.available?
        ensure_plurimath_loaded
        true
      rescue LoadError
        false
      end

      # Get Plurimath version
      #
      # @return [String] Plurimath version string
      #
      def self.plurimath_version
        ensure_plurimath_loaded
        Plurimath::VERSION
      end
    end
  end
end
```

### MathConfiguration

```ruby
# lib/uniword/math/math_configuration.rb

module Uniword
  module Math
    # Configuration for math equation handling.
    #
    # Responsibility: Manage configuration for math equation conversion,
    # formatting, and rendering options.
    #
    # Configuration is loaded from external YAML file.
    #
    class MathConfiguration
      attr_reader :default_display_mode, :default_alignment, :number_formatter

      # Initialize configuration
      #
      # @param config [Hash] Configuration hash
      #
      def initialize(config = {})
        @default_display_mode = config.fetch(:default_display_mode, :inline)
        @default_alignment = config.fetch(:default_alignment, :left)
        @omml_options = config.fetch(:omml_options, {})
        @mathml_options = config.fetch(:mathml_options, {})
        @latex_options = config.fetch(:latex_options, {})

        # Number formatter configuration
        @number_formatter = build_number_formatter(config[:number_formatting])
      end

      # Load configuration from file
      #
      # @param config_file [String] Path to configuration YAML file
      # @return [MathConfiguration] Configuration object
      #
      def self.load(config_file = nil)
        config_file ||= default_config_path
        config = Configuration::ConfigurationLoader.load_file(config_file)
        new(config[:math] || {})
      end

      # Get OMML conversion options
      #
      # @return [Hash] OMML options
      #
      def omml_options
        @omml_options
      end

      # Get MathML conversion options
      #
      # @return [Hash] MathML options
      #
      def mathml_options
        @mathml_options
      end

      # Get LaTeX conversion options
      #
      # @return [Hash] LaTeX options
      #
      def latex_options
        @latex_options
      end

      private

      def build_number_formatter(config)
        return nil unless config

        # Create Plurimath number formatter if configured
        if PlurimathAdapter.available?
          Plurimath::NumberFormatter.new(
            config.fetch(:locale, :en),
            localizer_symbols: config.fetch(:symbols, {}),
            precision: config[:precision]
          )
        end
      end

      def self.default_config_path
        File.expand_path('../../../config/math_config.yml', __dir__)
      end
    end
  end
end
```

## Integration Workflows

### Reading DOCX with Math

```ruby
# Workflow: DOCX → Uniword → Math Equations

# 1. Parse DOCX file
document = Uniword::Document.open('document_with_math.docx')

# 2. Extract math equations
equations = document.paragraphs.flat_map do |para|
  para.children.select { |child| child.is_a?(Uniword::MathEquation) }
end

# 3. Convert to various formats
equations.each do |equation|
  puts "LaTeX: #{equation.to_latex}"
  puts "MathML: #{equation.to_mathml}"
  puts "AsciiMath: #{equation.to_asciimath}"
end

# 4. Modify equation
equation = equations.first
new_formula = Plurimath::Math.parse('x^2 + y^2 = r^2', :latex)
equation.formula = new_formula

# 5. Save back to DOCX
document.save('document_modified.docx')
```

### Writing DOCX with Math

```ruby
# Workflow: Create document with math equations

# 1. Create document
document = Uniword::Document.new

# 2. Add paragraph with inline equation
para = Uniword::Paragraph.new
para.add_text("The Pythagorean theorem states that ")

# Create inline equation from LaTeX
equation = Uniword::MathEquation.new(
  formula: Plurimath::Math.parse('a^2 + b^2 = c^2', :latex),
  display_mode: :inline
)
para.add_child(equation)

para.add_text(" for right triangles.")
document.body.add_child(para)

# 3. Add block equation
block_equation = Uniword::MathEquation.new(
  formula: Plurimath::Math.parse('\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}', :latex),
  display_mode: :block,
  alignment: :center
)
document.body.add_child(block_equation)

# 4. Save to DOCX
document.save('math_document.docx')
```

### HTML/MathML Export

```ruby
# Workflow: DOCX → HTML with MathML

# 1. Load document
document = Uniword::Document.open('document_with_math.docx')

# 2. Convert to HTML with MathML

# 3. Or convert math to LaTeX for KaTeX/MathJax
html = document.to_html(math_format: :latex)

# Output:

# <div class="math-display">$$\int_{0}^{\infty} e^{-x^2} dx$$</div>
```

```ruby
# spec/uniword/math/plurimath_adapter_spec.rb

RSpec.describe Uniword::Math::PlurimathAdapter do
  describe '.from_omml' do
    it 'parses simple OMML equation' do
      omml = '<m:oMath><m:r><m:t>x</m:t></m:r></m:oMath>'
      formula = described_class.from_omml(omml)
      expect(formula).to be_a(Plurimath::Math::Formula)
    end

    it 'raises error for invalid OMML' do
      expect {
        described_class.from_omml('invalid')
      }.to raise_error(ArgumentError)
    end
  end

  describe '.to_omml' do
    it 'converts Plurimath formula to OMML' do
      formula = Plurimath::Math.parse('x^2', :latex)
      omml = described_class.to_omml(formula)
      expect(omml).to include('<m:oMath')
    end
  end
end
```

### Integration Tests

```ruby
# spec/integration/math_equation_spec.rb

RSpec.describe 'Math Equation Integration' do
  it 'round-trips OMML through Plurimath' do
    original_omml = '<m:oMath>...</m:oMath>'
    equation = Uniword::MathEquation.new(omml: original_omml)
    regenerated_omml = equation.to_omml

    # Verify formula equivalence
    expect(regenerated_omml).to be_mathematically_equivalent_to(original_omml)
  end

  it 'converts between formats without loss' do
    latex = '\frac{x^2 + y^2}{2}'
    equation = Uniword::MathEquation.new(
      formula: Plurimath::Math.parse(latex, :latex)
    )

    # Round-trip through OMML
    omml = equation.to_omml
    new_equation = Uniword::MathEquation.new(omml: omml)
    new_latex = new_equation.to_latex

    expect(new_latex).to eq(latex)
  end
end
```

## Performance Considerations

### Lazy Loading

- Load Plurimath only when math operations are needed
- Cache parsed formulas
- Defer conversion until serialization time

### Optimization Strategies

1. **Batch Conversions:** Convert multiple equations in one pass
2. **Caching:** Cache OMML ↔ Formula conversions
3. **Selective Loading:** Load only math features when needed
4. **Memory Management:** Release formula objects after conversion

## Error Handling

### Graceful Degradation

```ruby
# When Plurimath is not available
def to_latex
  if PlurimathAdapter.available?
    @formula.to_latex
  else
    # Fallback: return OMML as-is or error message
    "[Math equation - Plurimath gem required for conversion]"
  end
end
```

### Error Types

1. **ParseError:** Invalid OMML/math syntax
2. **ConversionError:** Unsupported math constructs
3. **DependencyError:** Plurimath not installed
4. **ConfigurationError:** Invalid math configuration
