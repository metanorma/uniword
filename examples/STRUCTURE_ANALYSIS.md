## Builder API Mapping

The structure is implemented using these Builder API methods:

```ruby
# Front matter
b.add_paragraph(text, style: 'Title')
b.add_paragraph(text, style: 'Subtitle')
b.add_paragraph(text, style: 'Emphasis')
b.add_paragraph(text, style: 'Date')

# Headings
b.add_heading(text, level: 1)  # H1
b.add_heading(text, level: 2)  # H2

# Body text and quotes
b.add_paragraph(text)
b.add_paragraph(text, style: 'Quote')
b.add_paragraph(text, style: 'Intense Quote')

# Blank lines (for spacing)
b.add_blank_line

# Tables
b.add_table do
  row do
    cell text, bold: true
    cell text
  end
end
```
